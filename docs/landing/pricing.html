Done-For-You

How the Business plan works

The Process

  1. Discovery call (30 min) — we learn your business, team size, existing tools, and what you want agents to do.
  2. Server setup (Day 1–2) — we deploy FaiykeOS on your server or VPS, configure the environment, and run migrations.
  3. Memory seeding (Day 2–3) — your business facts, client data, preferences, and SOP documents loaded into memory.
  4. Agent configuration (Day 3–4) — agents tuned to your business context, namespace branding applied.
  5. Training session (Day 5) — 1-hour recorded walkthrough with your team covering daily workflows.
  6. Launch & 30-day support — you go live. We handle any issues for 30 days at no extra charge.

What You Need to Provide

  • Server or VPS access (we can advise on hosting if needed)
  • Anthropic API key (from console.anthropic.com)
  • Business information to seed into memory
  • Brand colours and company name (optional)
  • SMTP credentials for email notifications (optional)

Optional Ongoing Support ($147/mo)

After the 30-day included support window, upgrade to a monthly support plan covering: agent prompt updates, memory maintenance, new workflow creation, user management, and priority response.
