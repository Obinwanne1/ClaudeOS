FAQ

Common questions

Do I need my own Claude / Anthropic API key?

  • Yes — FaiykeOS calls the Anthropic API using your key
  • No markup — you pay Anthropic directly at their standard rates (~$0.003/1k tokens for Sonnet)
  • Full control — set your own usage limits in the Anthropic console

What server do I need?

  • Minimum: 4 GB RAM, 4 CPU cores, stable internet
  • OS: Windows 10/11 or Linux (Ubuntu 20+)
  • No Docker, no Kubernetes — Python + SQLite only
  • A basic $10/mo VPS works fine for most teams

Can I customise the agents?

  • YAML files — agent behaviour is plain-text system prompts, no coding needed
  • Developers can extend the executor, add tools, or build new agent types
  • Temperature, max tokens, tool access — all configurable per agent

Is this a SaaS or a one-time purchase?

  • One-time purchase — no recurring platform fees
  • Only ongoing cost: Anthropic API usage (pay-per-token)
  • Optional $147/mo support plan available for Done-For-You buyers after launch

Can I deploy this for multiple clients?

  • Agency License covers unlimited client deployments under your brand
  • Each client gets isolated namespace: own memory, branding, users, and agents
  • SKILL.md build playbook automates each new deployment in minutes

What happens after I buy?

  • Self-Install: full source code + docs delivered immediately
  • Done-For-You: discovery call scheduled within 24 hours
  • Agency: code + license + SKILL.md playbook within 24 hours

Is the source code obfuscated or encrypted?

  • No — full, readable Python source code
  • Read, modify, extend, and maintain every line
  • It's your codebase once you purchase — no licence checks, no call-home
