FAQ

Common questions

Do I need my own Claude / Anthropic API key?

Yes. FaiykeOS uses your Anthropic API key for all agent inference. This means you control costs directly — you pay Anthropic only for what you use. There is no markup or platform fee on top of the Claude API cost.

What server do I need?

Any Windows 10/11 or Linux machine with 4GB RAM, 4 CPU cores, and a stable internet connection. A basic VPS or cloud VM works. The system uses Python + SQLite — no Docker, no Kubernetes, no managed cloud services required.

Can I customise the agents?

Yes, fully. Agent behaviour is controlled by YAML files with plain-text system prompts. No coding required to customise agents. Developers can extend the executor, add tools, or build entirely new agent types.

Is this a SaaS or a one-time purchase?

It's a one-time purchase for the code. There are no recurring platform fees. The only ongoing cost is your Anthropic API usage (pay-per-token) and optionally a support plan for the Done-For-You tier.

Can I deploy this for multiple clients?

Yes — the Agency License covers unlimited client deployments under your brand. Each client gets a fully isolated namespace with their own memory, agents, branding, and users. The SKILL.md build playbook automates each deployment from scratch in minutes.

What happens after I buy?

Self-Install: you receive the full source code and documentation immediately. Done-For-You: we schedule a discovery call within 24 hours to begin setup. Agency: you receive the code, license, and SKILL.md build playbook within 24 hours.

Is the source code obfuscated or encrypted?

No. You receive the full, readable Python source. You can read, modify, extend, and maintain every line. It's your codebase once you purchase.
